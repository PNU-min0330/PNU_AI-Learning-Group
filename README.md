## 📌 프로젝트 소개

**2026 PNU AI학습공동체**는 부산대학교 정보의생명공학대학 SW전공 학생들의 자기주도적 학습 능력과 AI·SW 전공역량 강화를 위한 자율활동 지원 프로그램입니다. 본 저장소는 해당 프로그램의 **모집 안내 페이지**와 **온라인 신청 시스템**을 제공합니다.

### ✨ 주요 특징

- 🎨 **모던 다크 테마** - AI·SW 전공생에게 어울리는 코드 에디터 분위기
- 📱 **반응형 디자인** - PC, 태블릿, 스마트폰 모두 지원
- ✍️ **5단계 신청 폼** - 단계별 진행으로 부담 없는 신청 경험
- 💾 **자동 데이터 관리** - Google Sheets & Drive 자동 저장
- 🚀 **GitHub Pages 배포** - 별도 서버 없이 무료 호스팅

## 📂 프로젝트 구조

```
PNU_AI-Learning-Group/
├── index.html              # 메인 페이지 (프로그램 안내)
├── apply.html              # 신청 페이지 (5단계 신청 폼)
├── AppsScript_코드.gs      # Google Apps Script 코드 (데이터 처리)
├── 설정가이드.html         # 비전공자용 설정 안내서
└── README.md               # 본 파일
```

### 📄 파일별 역할

#### `index.html`
프로그램 소개, 운영 일정, 지원 내용, 신청 자격, 추천 경진대회, 문의처를 안내하는 메인 페이지입니다.

#### `apply.html`
5단계로 구성된 온라인 신청 폼입니다:
1. **팀 정보** - 팀명, 프로젝트명, 참가 대회, 프로젝트 요약
2. **팀장 정보** - 인적 사항 및 연락처
3. **팀원 정보** - 최소 3명, 최대 5명까지 추가 가능
4. **서류 업로드** - 참가신청서 및 활동계획서 (PDF/HWP/DOCX, 최대 10MB)
5. **개인정보 동의** - 수집·이용, 제3자 제공, 서약 사항 동의

#### `AppsScript_코드.gs`
신청 데이터를 Google Sheets와 Google Drive에 자동 저장하는 백엔드 코드입니다.

## 🚀 시스템 아키텍처

```
   [사용자 웹브라우저]
          ↓
   [GitHub Pages]
   (정적 웹사이트 호스팅)
          ↓
   [사용자가 신청 폼 작성 + 파일 업로드]
          ↓
   [Google Apps Script]
   (중간 다리 역할)
          ↓
     ┌────┴────┐
     ↓         ↓
[Google Sheets] [Google Drive]
   신청 내역      업로드 파일
```

### 데이터 흐름

1. 사용자가 신청 페이지에서 정보 입력 및 파일 첨부
2. 파일은 Base64로 인코딩되어 JSON에 포함
3. Apps Script가 POST 요청을 받아 처리
4. 파일은 Drive의 팀별 폴더에 저장
5. 신청 내역은 Sheets에 한 줄로 추가
6. 사용자에게 완료 화면 표시

## ⚙️ 설정 방법

자세한 설정 방법은 [`설정가이드.html`](./설정가이드.html) 파일을 참고하세요.

### 빠른 설정 (5단계)

1. **Google Sheets 생성** → 시트 ID 복사
2. **Google Drive 폴더 생성** → 폴더 ID 복사
3. **Apps Script에 코드 붙여넣기** → ID 2개 입력
4. **웹앱으로 배포** → URL 복사
5. **`apply.html`의 `APPS_SCRIPT_URL` 변수에 URL 입력**

### ⚠️ 주의 사항

- Apps Script 배포 시 **액세스 권한**을 반드시 **"모든 사용자"**로 설정
- 코드 수정 후에는 반드시 **재배포** (배포 → 배포 관리 → 새 버전)
- `apply.html`의 `APPS_SCRIPT_URL`은 본인의 배포 URL로 교체 필수
