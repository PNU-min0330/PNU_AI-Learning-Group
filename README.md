## 배포 방법

### 1단계 — Google 스프레드시트 & 드라이브 폴더 준비

1. [Google Sheets](https://sheets.google.com)에서 새 스프레드시트 생성
- URL에서 **스프레드시트 ID** 복사
- 예: `https://docs.google.com/spreadsheets/d/`**`여기가ID`**`/edit`
1. [Google Drive](https://drive.google.com)에서 첨부파일 저장용 폴더 생성
- 폴더 열기 → URL에서 **폴더 ID** 복사
- 예: `https://drive.google.com/drive/folders/`**`여기가ID`**

-----

### 2단계 — Google Apps Script 배포

1. [script.google.com](https://script.google.com) → 새 프로젝트 생성
1. `Code.gs` 내용 붙여넣기 후 상단 `CONFIG` 수정

```javascript
const CONFIG = {
  SPREADSHEET_ID: '스프레드시트 ID',
  SHEET_NAME: '신청내역',
  FOLDER_ID: '드라이브 폴더 ID',
  NOTIFY_EMAIL: '담당자 이메일',
};
```

1. 배포 → 웹 앱 → 액세스 권한: **모든 사용자** → 배포
1. 발급된 웹 앱 URL을 `index.html`의 `SCRIPT_URL`에 붙여넣기

```javascript
const SCRIPT_URL = 'https://script.google.com/macros/s/...../exec';
```

-----

### 3단계 — GitHub Pages 배포

```bash
git init
git add index.html README.md
git commit -m "feat: 인턴십 신청 웹페이지 배포"
git remote add origin https://github.com/[계정명]/[저장소명].git
git push -u origin main
```

GitHub 저장소 → **Settings** → **Pages**  
→ Branch: `main` / `/ (root)` → **Save**

배포 URL: `https://[계정명].github.io/[저장소명]/`

## 동작 흐름

```
학생이 신청서 작성 및 PDF 파일 첨부
        ↓
index.html → fetch POST → Apps Script
        ↓
  ├── 구글 시트에 행 추가 (접수번호 자동 채번, 타임스탬프, 전체 필드)
  ├── PDF → 구글 드라이브 폴더 업로드 (공유 링크 시트에 기록)
  ├── 담당자 알림 이메일 발송
  └── 신청자 접수 확인 이메일 자동 발송
        ↓
화면 하단 토스트 알림 "신청 완료"
```
